# List all available ferry routes

List all available ferry routes

## Usage

``` r
tm_ferry_routes()
```

## Value

A sorted character vector of ferry route IDs available for stop lookup.

## Examples

``` r
tm_ferry_routes()
#> [1] "Boat-EastBoston" "Boat-F1"         "Boat-F4"         "Boat-F6"        
#> [5] "Boat-F7"         "Boat-Lynn"      
```
